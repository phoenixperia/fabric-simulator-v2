# Projection Texture Specification
# FIERO Fabric Simulator V2

---

# Overview

This document defines the projection and texture rendering rules for FIERO Fabric Simulator V2.

The system must preserve realistic fabric appearance and luxury tailoring presentation using real fabric photos.

---

# Texture Workflow

Official workflow:

1. Upload Fabric Photo
2. Crop Fabric Area
3. Scale Adjustment
4. Seamless Texture Generation
5. Repeat Texture Rendering
6. Mask Clipping
7. Final Render

---

# Fabric Upload Rules

Fabric photos may include:

- table background
- fabric folds
- lighting variation
- reference objects

---

# Important Rules

- Clean usable fabric area must be selected
- Realistic fabric detail must remain preserved
- Unwanted objects must be excluded

---

# Crop Rules

Fabric crop selection must preserve:

- clean fabric structure
- realistic texture detail
- stable pattern readability

---

# Important Rules

- Strong wrinkles should be avoided
- Distorted fabric regions should be avoided
- Manual crop control has priority

---

# Scale Adjustment Rules

Texture scale must preserve realistic fabric proportions.

repeat rendering
tile rendering
