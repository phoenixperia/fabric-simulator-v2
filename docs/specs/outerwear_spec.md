# Outerwear Specification
# FIERO Fabric Simulator V2

---

# Overview

This document defines the official OUTER rendering specifications used throughout FIERO Fabric Simulator V2.

OUTER garments render above MID tailoring layers while preserving:

- silhouette continuity
- overlap readability
- collar hierarchy
- projection continuity
- structural stability

The mannequin body remains fixed.

OUTER garments expand outward from MID structure.

---

# Official OUTER Layer Definition

OUTER garments render above MID garments.
