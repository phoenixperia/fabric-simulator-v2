# Naming Convention Specification
# FIERO Fabric Simulator V2

---

# Overview

This document defines the official naming conventions used throughout FIERO Fabric Simulator V2.

All renderer systems must follow standardized naming rules.

---

# Official Naming Structure

```txt
category_variant_view_version_resolution.extension
```

---

# Example

```txt
single_open_navy_front_v001_4k.png
```

---

# Separator Rules

Use:

```txt
underscore _
```

between all naming components.

---

# Important Rules

- Spaces are prohibited
- Hyphen mixing is prohibited
- Naming order must remain consistent

---

# Letter Rules

Use:

```txt
lowercase only
```

---

# Important Rules

- Mixed capitalization is prohibited
- Random abbreviations are prohibited

---

# Version Rules

## Official Format

```txt
v001
v002
v003
```

---

# Important Rules

- Production versions must never be overwritten
- Version numbering must remain sequential

---
