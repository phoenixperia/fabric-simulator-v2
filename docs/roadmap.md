# FIERO Fabric Simulator V2 Roadmap

## Purpose

This document defines the official development roadmap for FIERO Fabric Simulator V2.

The roadmap clarifies:

* Current development scope
* Future expansion plans
* Feature priorities
* Phase boundaries

The purpose is to maintain focused development while preserving realistic rendering quality and stable renderer architecture.

---

# Phase1 — Mens Core Renderer

## Goal

Build the foundational realistic mens tailoring renderer.

---

## Included Features

* Single Open Jacket
* Single Closed Jacket
* Double Closed Jacket
* Vest
* Pants
* Shirt
* Tie

---

## Core Systems

* Layer rendering system
* Independent mask system
* Repeat texture rendering
* Shadow multiply rendering
* Outline overlay rendering
* Basic background rendering

---

## Texture System

* Manual fabric crop
* 100 yen scale reference
* Simple seamless texture generation
* Repeat rendering

---

## UI Scope

* Fabric upload
* Layer visibility toggle
* Shirt selection
* Tie selection
* Background selection

---

## Phase1 Priorities

Highest priorities:

1. Realistic appearance
2. Stable rendering
3. Natural overlap behavior
4. Realistic tailoring balance
5. Fast customer presentation

---

## Not Included

The following features are intentionally excluded from Phase1:

* Women templates
* Skirt templates
* AI automatic crop
* Public customer editing UI
* Real-time 3D rendering

---

## Goal

Improve fabric workflow and scalability.

---

## Planned Features

---

# Long-Term Vision

The final goal is to create a realistic and elegant tailoring simulation system using real fabric photos.

The system prioritizes:

* Realistic appearance
* Tailoring balance
* Stable rendering
* Modular architecture
* Practical sales workflow

All future expansion must preserve the same rendering philosophy.
